# Bucas Quest

A 2D game built in Ruby where you play as "the bucas" on a quest. A personal project for learning game development with the Gosu library.

## Built with

- Ruby
- Gosu, a 2D game development library
- gosu_tiled, for loading Tiled tilemaps into Gosu

## Running it

1. Install the gems:
```
   gem install gosu gosu_tiled
```
2. Run the game:
```
   ruby main.rb
```

## Project structure

- `main.rb`: entry point
- `src/`: game source code
- `lib/`: supporting libraries
- `plan/`: design and planning notes
