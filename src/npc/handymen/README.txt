Name:
  Stendhal Handymen
Version:
  1.1
Description:
  Handymen sprites created for Stendhal as drop-in replacements. Based
  on work by Svetlana Kushnariova (Cabbit).
Details:
  - Dimensions: 24x32 & 48x64 (scale2x filter)
  - Orientation: orthogonal (N/E/S/W)
  - Animation:
    - idle (center frames)
    - walking: 12 frames (3 per direction)
  - PNG images use indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
